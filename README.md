## Pre-processadores SASS 

Para verificar o pre processador e caso queira realizar novas alterações é necessário: 

```bash
1. OU: Instalar o sass globalmente `npm i -g sass`
2. OU: Instalar extensão no VSCode `live sass compiler`
```

### Como executar o projeto

Finalizando todos os passos descritos, executar o projeto em um dos dois modos:

1) via cmd
```bash
# Acesse a pasta do projeto pelo terminal/cmd
$ cd dnaForMarketing

# Execute a aplicação
$ npx lite-server

# O servidor inciará na porta:3000 - acesse http://localhost:3000
```

2) Instalar Exntensão Live Server
- clicar com botão direito no arquivo .html na pasta do projeto 
```bash
open with Live Server
```


#### Observações

Caso ao executar o comando abaixo der erro, será necessário intalar o node na máquina.
```bash
npx lite-server
```
